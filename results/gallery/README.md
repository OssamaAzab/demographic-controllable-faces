# Full gallery

One preview per (method, identity, prompt): 1680 images, 512 px JPEG.
Layout is `<method>/<identity>/<prompt_key>.jpg`. The best of 5 seeds is shown,
chosen by detected-face + highest AdaFace identity; selection does not optimize
for control, so control successes and failures are both visible. Full-resolution examples are in
`results/samples/`. All on SDXL 1.0 + fp16-fix VAE (HyperLoRA on RealVisXL v4.0).
